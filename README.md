# BeadCraft Studio：拼豆图纸生成工具

## 1. 项目简介

BeadCraft Studio 是一个面向拼豆用户的网页端图纸生成工具。

用户上传一张像素图或普通图片后，系统会自动进行像素化处理，并将图片颜色匹配到真实拼豆品牌色盘中，生成带有网格线、色号标记和材料消耗清单的拼豆实体图纸。

本项目支持 PNG 图纸导出、PDF 打印版导出和 CSV 材料清单导出，用户可以根据生成结果直接准备拼豆材料并进行实体制作。

## 2. 功能特性

- 图片上传与原图预览
- 自动像素化处理
- 支持 16 × 16、24 × 24、32 × 32、48 × 48、64 × 64、96 × 96 等图纸尺寸
- 基于真实拼豆色盘进行最近色匹配
- 显示颜色匹配误差等级
- 支持透明背景处理
- 支持最大颜色数限制
- 支持网格线显示
- 支持色号显示
- 支持圆珠效果预览
- 自动统计材料清单
- 支持导出 PNG 图纸
- 支持导出 PDF 打印版
- 支持导出 CSV 材料清单
- 支持 localStorage 保存用户上一次设置
- 支持水平镜像
- 支持导出时自定义图纸标题
- 支持导出图显示坐标数字

## 5. 使用说明
打开网页。
上传一张 PNG、JPG 或 JPEG 图片。
选择图纸尺寸，例如 32 × 32。
选择是否忽略透明背景。
设置最大颜色数，例如不限、8 色、12 色、16 色或 24 色。
选择是否显示网格线、色号和圆珠效果。
点击生成拼豆图纸。
查看材料清单和色盘图例。
根据需要导出 PNG、PDF 或 CSV 文件。
## 6. 色盘数据来源说明
本项目使用真实拼豆品牌色号建立色盘数据。色盘数据文件位于：
src/data/hamaPalette.ts
项目中使用的 HEX 色值用于网页端颜色匹配和视觉展示。由于不同品牌、批次、屏幕显示和光照环境会造成颜色差异，实际制作时应以实体拼豆颜色为准。
## 7. 核心算法说明
系统首先使用 Canvas 将用户上传的图片缩放到指定宽高，得到目标拼豆图纸尺寸。
随后，系统读取缩放后图像的每一个像素点 RGB 值，并与拼豆色盘中的颜色逐一计算 RGB 欧氏距离。系统选择距离最小的颜色作为该像素对应的拼豆色号。
颜色距离计算逻辑如下：
distance = sqrt((r1-r2)^2 + (g1-g2)^2 + (b1-b2)^2)
误差等级划分：
颜色距离	等级	说明
0 - 40	low	匹配较准确
41 - 90	medium	有轻微色差
90 以上	high	色差较明显
如果用户开启透明背景处理，系统会根据 alpha 通道判断该像素是否为空格。当 alpha 小于设定阈值时，该格子不会生成拼豆，也不会进入材料清单。
如果用户设置最大颜色数，系统会先统计当前图纸中每种颜色的使用数量，再保留使用最多的若干颜色，并将低频颜色重新映射到保留颜色集合中，从而减少实际使用的拼豆颜色数量。

## 9. 导出说明
本项目支持三种导出方式：
### PNG 导出
用于导出拼豆图纸位图，适合直接查看、发送或打印。
### PDF 导出
用于生成打印友好的图纸文件，适合纸质制作参考。
### CSV 导出
用于导出材料清单，包含色号、颜色名称、HEX 值、使用数量、占比和平均颜色误差，方便用户统计和购买材料。

## 10. 项目亮点
本项目不仅完成了图片上传、色盘匹配和图纸导出的基础功能，还加入了多个面向真实使用场景的增强功能：

自动像素化普通图片；
透明背景处理，避免透明区域被误算成拼豆；
最大颜色数限制，降低材料准备成本；
颜色匹配误差提示，帮助用户判断替代颜色效果；
圆珠效果预览，使图纸更接近真实拼豆效果；
PNG、PDF、CSV 多格式导出；
localStorage 保存设置，提升使用体验；
水平镜像和坐标数字，方便实体拼豆制作。
